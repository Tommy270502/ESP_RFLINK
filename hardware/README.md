# Hardware

KiCad hardware source for Wireless Dev Bridge V1.

Open the KiCad project at:

```text
hardware/kicad/ESPxRF.kicad_pro
```

Manufacturing outputs are kept separately in:

```text
manufacturing/gerbers
```

See [Hardware Guide](../docs/hardware.md) for board assets, KiCad notes, manufacturing review steps, and firmware pin alignment.
